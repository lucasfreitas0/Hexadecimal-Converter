# 🔢 Base Converter

Aplicação desktop para conversão entre sistemas numéricos (Binário, Octal, Decimal e Hexadecimal), com suporte a conversão de texto ASCII.

---

## 🚀 Funcionalidades

- Conversão entre bases numéricas:
  - Binário
  - Octal
  - Decimal
  - Hexadecimal

- Conversão de texto para bases
- Conversão de bases para texto
- Interface gráfica simples (Tkinter)

---

## 🧠 Objetivo do projeto

Este projeto foi desenvolvido para praticar:

- Lógica de programação
- Manipulação de bases numéricas
- Conversão de ASCII
- Desenvolvimento de interface gráfica com Python

---

## 🛠️ Tecnologias

- Python 3
- Tkinter

---

## ▶️ Como executar

```bash
python interface.py
